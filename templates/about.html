Описание

PatternMaker - сервис для создания
изображений, называемых паттернами. Паттерны при стыковке друг к другу,
образовывают равномерное поле.

Использование паттернов

Обычно паттерны используют для украшения обоев, рисунков
и изображений. Но их, так-же можно использовать для бэкграундов сайтов
и рекламы.

Есть ли смысл в сервисе, если есть photoshop?

Конечно есть. Ведь при создании паттернов в photoshop, или
других фоторедакторах, появляется множество проблем, например выравнивание
изображений относительно друг друга, или их расстоновка так, что-бы
паттерны при стыковке образовывали равномерое поле.
